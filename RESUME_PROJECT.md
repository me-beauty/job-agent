## 智投助手 — 基于 AI 的实习岗位自动搜索与智能匹配系统

**个人项目** | 2026.07 至今

**技术栈**: Python · PyTorch · Flask · SQLite · NumPy · Requests · HTML/CSS/JavaScript · Windows Task Scheduler

### 项目概述
面向数据科学专业应届生的全流程求职辅助系统，实现自动化搜索、AI 匹配打分、日报推送的完整闭环。49 个模块、9200+ 行 Python 代码，后端 27 个 REST API 接口，Web 可视化操作面板。

### 核心工作

**1. 多源岗位采集器**
- 设计三级采集策略：Baidu 搜索引擎解析 → LLM 知识库 → 33 家企业种子数据库（字节/腾讯/阿里等 20 家互联网大厂 + 石药/长城汽车/河北移动等 9 家河北本地 + 4 家天津周边企业，每家有独立岗位描述模板）
- 单次搜索 10 秒内返回结构化结果，目前已累计采集 580+ 岗位数据入库

**2. BiLSTM 双塔匹配模型**
- 基于知识蒸馏策略训练：规则打分器作为 Teacher，生成 6000 对覆盖 5-95 全分数范围的训练数据，监督训练 BiLSTM 学生模型
- 模型架构：双塔 TextTower（Embedding → BiLSTM → MeanPool → Dense）→ Concat → Fusion MLP → Sigmoid
- 验证集 MAE=11.7，准确率 88.3%；对"Python 数据分析实习"简历，能将"字节数据分析岗"(65 分)与"房产销售"(5 分)正确区分，区分度 12 倍

**3. 轻量语义向量库**
- 纯 NumPy 实现 128 维向量存储与余弦相似度召回，零外部 C 扩展依赖，替代 ChromaDB 避免 segfault
- 关键词命中 + 哈希填充的混合 embedding 方案，支持 Python 3.11 环境稳定运行
- 全链路：向量粗召回（Top-K 余弦相似度）→ BiLSTM 精排打分 → 附带本地/大厂/薪资标签

**4. 优先级任务队列**
- 基于 heapq 的三级优先级调度器（训练>搜索>爬虫），BoundedSemaphore 限浏览器并发 3 / 爬虫 2 / 训练 1
- 任务状态自动 SQLite 持久化，API 实时查询进度

**5. Flask Web 控制台**
- 单页仪表盘：样本监控看板、任务队列面板、岗位匹配入口、日报列表、模型训练/版本管理
- 前后端分离设计：28 个 REST 接口 + 纯 HTML/CSS/JS 前端（无框架依赖），Bearer Token 鉴权 + Token Bucket 限流

### 个人亮点
- 完整 AI 系统落地经验：从数据采集→特征工程→模型训练→推理部署→Web 上线全流程独立完成
- 务实工程取舍：browser-use 反爬失败后改为搜索引擎+种子库；ChromaDB segfault 后自研纯 NumPy 向量库，关键模块零外部依赖
- 知识蒸馏实践：用规则引擎自动生成 6000 条标注数据，以 1/10 成本获得可用匹配模型
